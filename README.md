# E-Commerce Sales: Data Cleaning Report

## Project Overview
Data cleaning project on a raw e-commerce dataset of 1,200 orders 
to ensure data integrity and prepare it for analysis.

## Tools Used
- Microsoft Excel — Data cleaning and documentation

## Dataset
- Total Records: 1,200
- Total Columns: 14
- Date Range: January 2023 — June 2025

## Cleaning Steps
1. Checked and confirmed zero duplicate Order IDs
2. Checked and confirmed zero duplicate Tracking Numbers
3. Replaced 309 blank Coupon Code values with N/A
4. Reformatted all 14 columns from General to correct data types
5. Verified Total Price accuracy (Quantity × Unit Price)

## Data Quality Results
- Duplicate Order IDs: 0
- Duplicate Tracking Numbers: 0
- Missing Values Resolved: 309
- Column Formatting: 14 columns corrected
- Total Price Accuracy: 100% verified

- E-commerce sales dataset.xlsx — Contains Raw Data, 
  Cleaned Data and Cleaning Summary sheets
